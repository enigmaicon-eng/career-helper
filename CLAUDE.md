# Career Helper — Career Tasks Layer

This repository provides the **career tasks** layer for the integrated career agent system.

## Role in the System

| Layer | Repository | Purpose |
|---|---|---|
| **Thinking** | `superpowers` | Planning, TDD, debugging, brainstorming |
| **Strategy** | `agents` | Multi-agent coordination, PM frameworks |
| **Career Tasks** | `career-helper` (this repo) | Job search, CV, interviews, coaching |

## Available Skills (`career-helper:*`)

### Start Here
- `career-helper:tim` — guided career coaching; Tim routes you through the right skills in order
- `career-helper:getting-started` — orientation and how to use career-helper

### Job Search & Applications
- `career-helper:career-navigator` — job search planning, networking, salary negotiation, offer evaluation
- `career-helper:application-optimiser` — ATS-optimised CV rewriting, company research, application strategy

### Interviews
- `career-helper:interview-master` — interview prep, mock interviews, rejection debrief, ageism handling

### Online Presence
- `career-helper:linkedin-coach` — LinkedIn profile audit, headline crafting, content strategy
- `career-helper:social-media-review` — social media audit for employer-readiness
- `career-helper:employer-footprint` — digital footprint check

### Career Direction
- `career-helper:career-transitions` — fractional work, portfolio careers, founder path, AI readiness
- `career-helper:ai-impact-assessment` — AI threat/opportunity assessment for your role

### Specialist
- `career-helper:ned-ai-helper` — AI governance for Non-Executive Directors and board members

## Workflow

1. Start with `career-helper:tim` for guided coaching, or pick a skill directly
2. Use `superpowers:brainstorming` before any planning work
3. Use `agents:team-coordination` when multi-perspective analysis is needed

## Commands

- `/career-helper:career-coach` — launch Tim, your career coach
- `/career-helper:quick-start` — quick start guide
- `/career-helper:help` — list all available skills
- `/career-helper:status` — show session progress
